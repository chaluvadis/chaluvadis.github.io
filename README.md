## Content Philosophy

This portfolio represents how I think about systems, trade-offs, and software evolution—not just a list of technologies or projects.

**Key sections:**
- Production systems with architecture depth + AI justification
- Developer tooling as augmentation to platform work
- Explicit examples of when NOT to use AI
- Honest trade-offs and constraints

## Tech Stack
- Content: Semantic HTML5
- Styling: Minimal CSS with dark mode support
- Deployment: GitHub Pages (static site, no build step)
- Philosophy: Content over chrome

## Local Development
```bash
# No build system needed - just open index.html
open index.html

# Or use any static server:
python -m http.server 8000
# Then visit http://localhost:8000
```

## Design Principles
- Single-column layout (max 900px)
- Typography-driven hierarchy
- No frameworks, no JavaScript
- Reads like a technical document
- Dark mode via prefers-color-scheme
