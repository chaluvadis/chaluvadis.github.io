# about

Staff-level platform & backend engineer with 15+ years building infrastructure that helps teams ship software with less friction. Backend APIs, developer tooling, platform workflows, workload identity, and deployment systems.

Most of my projects start as side experiments — I scratch my own itch, automate something tedious, or build a tool to make my own workflow smoother. If it saves me time, I polish it and open-source it. If it helps someone else along the way, even better.

My focus areas are zero-trust infrastructure, developer experience, and cloud operations. I work across Kubernetes, .NET, TypeScript, Go, and Python.
